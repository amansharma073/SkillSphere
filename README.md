# SkillSphere 🚀

A modern **skill-sharing platform** where users can explore, add, and showcase skills. Built with React and Tailwind CSS, focusing on clean UI and responsive design.

---

## 🌐 Live Demo

👉 https://skillsphere-one-ivory.vercel.app/

---

## 📌 Features

* 🔍 Explore skills shared by users
* ➕ Add your own skills
* 👤 User profile section
* 🏷️ Badge-based UI system
* ⚡ Fast and responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **State Management:** Context API
* **Build Tool:** Vite

---

```
src/
 ├── components/

 ├── data/
 └── utils/
```

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/amansharma073/SkillSphere.git
```

2. Navigate to project folder

```bash
cd SkillSphere
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm run dev
```
